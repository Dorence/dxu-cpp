# dxu-cpp

A C++ utility library providing common helpers and data structures.

## Features

- **Slice**: A lightweight, non-owning reference to a byte array (similar to `std::string_view` but with more utility functions).
- **Conversion**:
  - Safe string-to-integer conversions (`StrToIntSafe`, `StrToInt64Safe`).
  - Human-readable byte size formatting (`HumanReadableBytes`).
  - String trimming (`TrimString`).
  - String to vector parsing (`StringToVectorInt`).

## Requirements

- C++11 or later, best for C++17
- CMake 3.20 or later

## Build

```bash
mkdir build
cd build
cmake ..
make
```

## Test

```bash
cd build
make tests
test/*_test
```

## Integration

- Prepare: (auto generated by cmake) generate `dxu/version.h` based on `dxu/version.h.in`.
- Declaration: include `include` directory.
- Impl All in One: `#include "dxu/impl.h"` in any source file. Don't include duplicatedly.
- Impl Partial: define `DXU_*_IMPLEMENTATION` macros before including `dxu/*.h`.

### CMake Options

- `DXU_NAMESPACE`: Namespace for the library (default: `dxu`).

### Single-File Library Rules

- File structure:

```cpp
#ifndef DXU_???_H_INCLUDE
#define DXU_???_H_INCLUDE

/* === declaration part === */

#endif  // DXU_???_H_INCLUDE
#ifdef DXU_???_IMPLEMENTATION

/* === implementation part === */

#endif  // DXU_???_IMPLEMENTATION
```

- `__INTELLISENSE__` is added to help VS/VSCode intellisense.
